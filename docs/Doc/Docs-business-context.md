# SkyPoint Financial Solutions – Business Context

SkyPoint Financial Solutions is a fictional small financial services company that provides bookkeeping, tax preparation, and financial consulting for small businesses.

They handle sensitive financial documents such as bank statements, payroll reports, and tax filings.

## Cloud Requirements
- Secure storage of financial data
- Clear separation of environments (dev vs prod)
- Strong access control
- Centralized monitoring
- Cost visibility
- Ability to scale as the business grows

## Why Azure
Azure provides built-in security, governance, monitoring, and identity tools that support financial data protection.
